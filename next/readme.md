## Install

```sh
git clone https://github.com/filecoin-project/specs.git
cd next # until we move this top level
yarn install
```

## Develop

### Update submodules
```sh
git submodule update --init
```

### Serve with Live Reload
```sh
yarn serve
# open http://localhost:1313/ in the browser
```
